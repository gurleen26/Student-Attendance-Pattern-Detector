# 🎓 Student Attendance Pattern Detector  

A DSA-based project that detects student attendance patterns using **stacks** and **sets**, with **JSON for data storage**, **JFrame for GUI**, and **Maven for build management**.  

---

## 📖 Description  
The **Student Attendance Pattern Detector** helps analyze student attendance records and detect patterns such as:  
- Consecutive absences  
- Repeated attendance patterns  
- Consistency across sessions  

This project demonstrates how **Data Structures and Algorithms (DSA)** can be combined with modern tools like **JSON, Maven, and Java Swing (JFrame)** to build a practical application.  

---

## 🚀 Features  
- 📂 Attendance data stored in **JSON files**  
- 📊 Pattern detection using **stacks** and **sets**  
- 🖥️ User-friendly interface built with **JFrame**  
- ⚡ Built and managed with **Maven**  

---

## 🛠️ Tech Stack  
- **Java** (Core + Swing)  
- **JSON** (for data storage)  
- **Maven** (build tool)  
- **DSA Concepts**: Stacks & Sets
